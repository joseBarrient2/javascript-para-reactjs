Javascript para React JS

1. Template literales

Conocidos tambien como plantillas literales. Habilitan que se pueda agregar variables en la cadena de texto, estas variables pueden ser usadas solo dentro de las comillas invertidas (``).

2. Short Hand Property Names

Es una forma sencilla de crear un objeto de forma literal asignadole variables y no haciendolo con una clave y valor. Solo haciendo referencia a la variable. Esto ayuda a escribir menos código

3. Arrow function

Es una nueva forma de definir funciones Esta sintaxis no ayuda a escribir una funciones mas cortas.

4. Destructuring

Esta expreción no ayuda a desempaquetar los valores de arreglos y objetos en distintas variables.

5. Parameters defaults

Los parametros por defecto son la forma como tratamos a las variables o argumentos que no se envian. Se les asigna un valor en caso no esta no lo tenga.

6. Es Modules

Es la exportacion e importacion de variables un funciones es una característica que nos permite cargar código desde diferentes archivos.

7. perador ternario

Es una alternativa al if/else tradsiciona, más comprimido simple y fácil de leer.
